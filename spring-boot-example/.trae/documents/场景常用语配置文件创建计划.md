# 场景常用语配置文件创建计划

## 任务概述
在 Spring Boot 项目中创建一个新的 YAML 配置文件，用于存放不同场景及其对应的 10 条常用语。

## 配置文件设计

### 文件位置
`src/main/resources/scene-phrases.yml`

### 文件结构
```yaml
scenes:
  - sceneId: 1
    scene: '沙滩'
    list:
      - '快看！那边的浪花特别大'
      - '记得涂好防晒霜，这儿的紫外线非常强'
      - '这片沙滩的沙子真细，踩上去像踩在棉花上一样。'
      - '咱们在沙滩上打场排球怎么样'
      - '...'
      - '...'
      - '...'
      - '...'
      - '...'
      - '...'
  - sceneId: 2
    scene: '...'
    list:
      - ...
```

## 实施步骤

### 步骤 1: 创建配置文件
- 在 `src/main/resources/` 目录下创建 `scene-phrases.yml` 文件
- 定义配置文件结构，包含 scenes 列表
- 为每个场景添加 sceneId、scene 和 10 条常用语

### 步骤 2: 添加示例场景数据
- 添加 1-2 个示例场景（如：沙滩、餐厅等）
- 每条常用语使用中文

## 预期输出文件
- `src/main/resources/scene-phrases.yml` - 包含场景常用语配置的 YAML 文件
